This file reads in parameters for the onset of each condition annually in the CMD module in the TLO model. Each tab represents one condition modelled.

3

baseline_annual_probability has been calibrated for each condition to data from Malawi. All other parameters are from Professor Ed Gregg’s group. Where possible, sources are listed to the right of the parameter values.
